## Overview
NexaConnect is an enterprise API management platform that enables organizations to design, publish, secure, and monitor APIs across internal and external consumers.

---

## Key Features
- API gateway and proxy
- Developer portal
- API analytics
- Rate limiting and throttling
- OAuth 2.0 and JWT support
- API versioning
- GraphQL support
- Service mesh integration
