## Overview
NexaInsight is an AI-powered business intelligence platform that allows business users to explore data, generate insights, and build dashboards without SQL or coding knowledge.

---

## Key Features
- Natural language query (ask data questions in plain English)
- AI-generated insights and anomaly alerts
- Drag-and-drop dashboard builder
- 350+ chart types
- Scheduled reports
- Mobile-responsive dashboards
- Embedded analytics SDK
